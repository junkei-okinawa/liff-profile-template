# 02. ユースケース図

```mermaid
graph LR
    User((ユーザー))

    subgraph "LIFF Application"
        UC1(ログインする)
        UC2(プロフィールを閲覧する)
        UC3(利用規約を確認する)
        UC4(利用規約に同意する)
        UC5(ログアウトする)
        UC6(退会する)
    end

    subgraph "System/Backend"
        UC_Sys1(同意日時を記録する)
        UC_Sys2(ユーザー情報を削除する)
    end

    User --> UC1
    User --> UC2
    User --> UC3
    User --> UC5
    User --> UC6

    UC3 -.->|未同意の場合| UC4
    UC4 -->|Firestore更新| UC_Sys1
    UC6 -->|将来的な実装| UC_Sys2
    UC6 -.->|完了後自動ログアウト| UC5
```

## ユースケース記述

| ID | ユースケース名 | 概要 |
| :--- | :--- | :--- |
| UC1 | ログインする | LINEアカウントを使用してアプリにログインする。 |
| UC2 | プロフィールを閲覧する | 自身のアイコン、名前、ID等を確認する。 |
| UC3 | 利用規約を確認する | アプリの利用規約を閲覧する。 |
| UC4 | 利用規約に同意する | 規約に同意し、そのステータスをシステムに保存する。 |
| UC5 | ログアウトする | アプリからログアウトし、セッションを終了する。 |
| UC6 | 退会する | サービスの利用を停止し、退会処理（ログアウト含む）を行う。 |
